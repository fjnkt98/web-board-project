## Environments

- Python 3.14
- Django 6.0
- uv 0.11.12

## Setup

Install [uv](https://docs.astral.sh/uv/getting-started/installation/#standalone-installer)

```sh
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Install Django

```sh
uv add django
```

Start project

```sh
uv run django-admin startproject apps .
```
